Essential requirements to run the program

OS Supported: Windows 10/11


Minimum Requirments Specification :

AMD Athlon Silver 3050u

RADEON Graphics

4GB RAM

5GB Disk Space

WebCam


Installation required:
- python 3.7 (Required!)
https://www.python.org/downloads/

Install the following lib with pip on cmd (recommended version):

  • OpenCV (incl. Numpy)
    pip install opencv-python

  • PILLOW
    pip install pillow

  • Tensorflow (incl. Keras and Tensorboard)
    pip install tensorflow for GPU CUDA or pip install tensorflow-cpu FOR CPU ONLY / AMD USER

  • FaceNet Keras model
    facenet_keras.h5

  • HaarCascade Classifier Models
    Haar Cascade

  • Python Local Library
    TkInter, Random, OS

Do's and Don'ts

Rules:
- one person per run
- open 'DAFTAR HADIR.xlsx' file for real time update every 1 min
- for the first user, register the face first before running attendance function
note: never open 'Attendance.csv' file on spreadsheet program (ex: Ms. Excel) when the program running

How to register face

There are two ways to register your face:

  • Open the main program (.exe)
  • click 'DAFTAR' button
  • enter the face name and your occupation (one person per runs)
  • click "AMBIL GAMBAR" to take photos
  • If all set, click the facecam window, press 'ENTER' on your keyboard to take image, and camera will automatically close and bring app back to main menu
  • Check if the photo is registered in the folder (default MyAbsence/Assets/data/)

How to use attendance program

!important: you need to register at least one face before you run this function.

  • Open the main program (.exe)
  • Click 'ABSEN' button
  • Remember the rules: one person per runs
  • If all set, click the facecam window,press 'ENTER' on your keyboard take absence, press 'ESC' on your keyboard to close camera, and camera will automatically close and bring app back to main menu
  • you can check at DAFTAR HADIR.xlsx it will updated every 1 min after you take absence

    • NOTE!:Be Patient is gold :) .
