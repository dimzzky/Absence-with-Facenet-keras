Cara Penggunaan

Hal - hal yang dibutuhkan untuk menjalankan program

OS yang didukung: Windows 10/11


Spesifikasi Minimum:

AMD Athlon Silver 3050u

RADEON Graphics

4GB RAM

5GB Disk Space

WebCam


Instalasi yang diperlukan:
- python >= 3.7
https://www.python.org/downloads/

Install library - library di bawah menggunakan pip di cmd:

  • OpenCV (Termasuk Numpy)
    pip install opencv-python

  • PILLOW
    pip install pillow

  • Tensorflow (Termasuk Keras and Tensorboard)
    pip install tensorflow for GPU CUDA or pip install tensorflow-cpu FOR CPU ONLY / AMD USER

  • FaceNet Keras model
    facenet_keras.h5

  • HaarCascade Classifier Models
    Haar Cascade

  • Python Local Library
    TkInter, Random, OS

Hal - hal yang harus diketahui sebelum menjalankan program

Ketentuan:
- Satu orang sekali menjalankan fungsi (baik registrasi maupun presensi).
- buka file 'DAFTAR HADIR.xlsx' untuk update secara real-time setiap 1 menit anda melakukan absensi.
- Untuk penggunaan pertama, sebelum menjalankan fungsi attendance, registrasikan setidaknya satu wajah terlebih dahulu.
catatan: jangan membuka file 'Attendance.csv' di program spreadsheet (seperti Ms. Excel) saat program berjalan.

Cara melakukan registrasi wajah

Ada dua cara melakukan registrasi wajah:

  • buka program utama (.exe)
  • klik tombol 'DAFTAR' button
  • masukkan nama dan pekerjaan(satu orang untuk sekali klik tombol face registration)
  • klik 'AMBIL GAMBAR' untuk ambil gambar
  • Jika sudah siap, silahkan klik window face cam, lalu tekan 'ENTER' di keyboard untuk mengambil gambar, dan kamera akan otomatis tertutup anda akan dibawa ke main menu kembali.
  • Melakukan pengecekan setelah foto ter-registrasi di folder (default MyAbsence/Assets/data/)

Cara menggunakan attendance program

!Penting: Anda harus registrasi setidaknya satu wajah sebelum menjalankan fungsi ini.

  • Buka program utama (.exe)
  • klik tombol 'ABSEN'
  • Ingat: sekali jalan satu orang
  • Jika sudah siap, silahkan klik window face cam,lalu tekan 'ENTER' di keyboard untuk mengambil absen, lalu tekan 'ESC' di keyboard untuk menutup kamera
  • Anda dapat memantau nama anda di DAFTAR HADIR.xlsx setelah 1 menit melakukan absensi.
